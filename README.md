# 项目特色
- **深度还原：** 深入考究传统塔罗，精准还原占卜过程每一个细节。
- **问题指定：** 支持向塔罗提问，占卜会根据你的提问进行，占卜结果将回答你的提问。
- **自研算法：** 全力提升占卜结果的可参考性，同一时间针对同一提问的占卜能够输出相同或相似的结果。
- **自动化占卜：** 自动洗牌、自动切牌、自动抽牌，提出问题即可一键占卜。
- **本地化支持：** 在做！
